# [Home](../) / Database

These are the database migration and seed scripts using Phinx and SQLite for the single-page application (SPA) skeleton based on Mithril.js and Slim Framework.


## Requirements

- PHP >= 8.0
- Composer >= 2.0


## Installation

    composer install


## Usage

Setup database

    composer db:setup

Refresh database

    composer db:refresh

Remove database

    composer db:remove
